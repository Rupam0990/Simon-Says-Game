## 🕹️ What is Simon Says?
Simon Says is a classic memory skill game. The game produces a sequence of colors and the player must repeat the sequence in the correct order. With each level, the sequence gets longer and harder to remember. How far can you go?

## 📖 How to Play
1. Press any key to start the game
2. Watch carefully which color button flashes — that is the sequence
3. Click the color buttons in the same order as they flashed
4. If you get it right, the game moves to the next level with one more color added to the sequence
5. If you click the wrong button, the screen flashes red and the game is over
6. Your highest score is tracked on screen
7. Press any key again to start a new game

## ✨ Features
- Random color sequence generation every level
- Visual flash feedback for both game and user clicks
- Red screen flash on wrong answer
- Level counter displayed on screen
- Highest score tracking
- Keyboard press to start/restart game
